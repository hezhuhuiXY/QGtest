# 前端子项目

本目录与后端 `D:\ideaproject\system` 同属一个 Git 仓库，统一推送到 [QGtest](https://github.com/hezhuhuiXY/QGtest.git)。

| 目录 | 说明 |
|------|------|
| `html-ts-admin` | 原生 HTML + JS/TS + CSS，可用 `node server.js` 本地预览 |
| `react-admin` | React + Vite，需 `npm install` 后 `npm run dev` |
| `vue3-element-admin` | Vue3 + Element Plus + Vite，需 `npm install` 后 `npm run dev` |

克隆仓库后请在各子目录内自行执行 `npm install`（`node_modules` 已加入 `.gitignore`，不会提交）。
